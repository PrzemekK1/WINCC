Locate Qt Compressed Help files (*.qch) for installed JCOP components.

For installed components, described by data points of type _FwInstallationComponents, get the path to all files listed in the .qtHelpFiles data point element. This function also verifies that the files exist and can be read.

)

Find pre-existing Qt Compressed Help files (*.qch)

Searches the 'help' directories in relevant locations, starting from the main WinCC OA installation directory, to find existing QCH files. This should locate the WinCC OA primary documentation.

Parameters
[out]qchFilePathsA list that QCH file paths will be appended to.
activeLangIdOaLanguage enum of the language subdirectory to search. By default this is undefined language (search directly in help/ and not in lang subdir)
searchPathMinMinimum search path level which is passed as the level parameter for the getPath function. Default is 1.
searchPathMaxMaximum search path level which is passed as the level parameter for the getPath function. Default is the constant SEARCH_PATH_LEN.
